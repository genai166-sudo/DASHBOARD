# 국내 주요 도시 날씨

Open-Meteo API + OpenStreetMap(Leaflet)을 활용한 대한민국 주요 8개 도시 실시간 날씨 페이지입니다.

**API Key · 토큰 불필요** — 바로 실행 가능합니다.

## 도시

서울 · 부산 · 인천 · 대구 · 대전 · 광주 · 울산 · 제주

## 실행

`weather/weather.html`을 브라우저에서 열면 됩니다.

## 사용 API

| 용도 | 서비스 |
|------|--------|
| 날씨 | [Open-Meteo](https://open-meteo.com/) |
| 지도 | [OpenStreetMap](https://www.openstreetmap.org/) + [Leaflet](https://leafletjs.com/) |
